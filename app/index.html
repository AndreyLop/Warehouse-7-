      About this project

      Following technologies and libraries where used in this project:

      • Routie - a lightweight router library which allows users to copy and share links to uploaded files, and save users history.
      • Handlebars - a templating engine which allowed me to generate main content, single item pages, gallery and create error messages.
      • bxSlider - a jQuery slider which works great on all browsers and supports touch slide change on mobile devices.
      • Featherlight - a lightweight light box for gallery of images.
      • jQuery - a JS library which provides easier, cross-browser support development.
      • SASS - a CSS preprocessor.
      • Node.js - server side JavaScript, its used to generate all info about uploaded file and handles file movement.
      • Express - Node.js framework.
